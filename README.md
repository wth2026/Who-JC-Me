# Who JC Me
保护你机房电脑的好工具

2025/8/20日，我红温了，因为机房有人一周内3天关了我7次电脑，我们这个电脑还带C盘还原，就非常烦人。于是，我突发奇想，写了这么一个项目，旨在保存程序停止运行前最后一段时间出现在你镜头之内的人。

Upd on 2025/8/21

目前功能：不停的拍照保存直到程序关闭，仅保存一段时间内的图片，以免消耗空间过大
